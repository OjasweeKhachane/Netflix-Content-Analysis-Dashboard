# Netflix-Content-Analysis-Dashboard

[View the interactive Tableau Dashboard] https://public.tableau.com/views/NetflixContentAnalysisDashboard_17809175631680/NetflixContentAnalysisDashboard?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

Developed an interactive Tableau dashboard to analyze Netflix content trends, ratings, country-wise distribution, and content growth over time using data visualization and storytelling techniques.

• Cleaned and transformed Netflix dataset using calculated fields and filters in Tableau.
• Created KPI cards to track total titles, movies, and TV shows.
• Built interactive visualizations including bar charts, trend analysis, and geographic maps.
• Implemented dynamic filters for content type, country, rating, and release year.
• Analyzed content distribution by country, rating category, and release trends.
